Preview Features are enabled organization-wide in Microsoft Defender XDR.

This setting is not available via Microsoft Graph API and requires manual verification.

#### Remediation action:

1. Open [Microsoft 365 Defender Portal](https://security.microsoft.com)
2. Navigate to **Settings > Microsoft Defender XDR > Preview Features**
3. Toggle Preview Features to on
4. Verify the setting matches the expected value: **On**

- [Microsoft 365 Defender Portal](https://security.microsoft.com)
