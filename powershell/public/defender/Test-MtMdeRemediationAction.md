Remediation action for all threat levels is set to Quarantine for consistent threat handling.

This setting is not available via Microsoft Graph API and requires manual verification.

#### Remediation action:

1. Open [Microsoft Endpoint Manager](https://endpoint.microsoft.com)
2. Navigate to **Endpoint Security > Antivirus > Threat Severity Default Action**
3. Set the remediation action for all threat severity levels to Quarantine
4. Verify the setting matches the expected value: **Quarantine**
