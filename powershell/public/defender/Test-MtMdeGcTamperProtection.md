Tamper Protection is enabled tenant-wide in Advanced Features.

This setting is not available via Microsoft Graph API and requires manual verification.

#### Remediation action:

1. Open [Microsoft 365 Defender Portal](https://security.microsoft.com)
2. Navigate to **Settings > Endpoints > Advanced Features > Tamper Protection**
3. Toggle Tamper Protection to on
4. Verify the setting matches the expected value: **Enabled**
