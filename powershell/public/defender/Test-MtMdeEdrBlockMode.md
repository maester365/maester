EDR in Block Mode is enabled for Microsoft Defender Antivirus devices.

This setting is not available via Microsoft Graph API and requires manual verification.

#### Remediation action:

1. Open [Microsoft 365 Defender Portal](https://security.microsoft.com)
2. Navigate to **Settings > Endpoints > Advanced Features > EDR in Block Mode**
3. Toggle EDR in Block Mode to on
4. Verify the setting matches the expected value: **Enabled**

- [Microsoft 365 Defender Portal](https://security.microsoft.com)
