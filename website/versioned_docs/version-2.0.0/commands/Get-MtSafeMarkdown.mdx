## SYNOPSIS

Escapes text to be safe to use in markdown.

## SYNTAX

```powershell
Get-MtSafeMarkdown [[-Text] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Escapes text to be safe to use in markdown.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-MtSafeMarkdown -Text $tenantName
```

Converts the content of $tenantName into something safe to use in markdown.
