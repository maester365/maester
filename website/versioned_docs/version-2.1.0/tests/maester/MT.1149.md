# Behavior Monitoring should be enabled

## Description

Verifies that behavior monitoring is enabled to detect advanced threats based on process behavior.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus > Real-time Protection. Ensure 'Allow Behavior Monitoring' is set to 'Allowed'.
