## SYNOPSIS

Counts enabled user accounts that have never logged on.

## SYNTAX

```powershell
Test-MtAdUserNeverLoggedInCount [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

This test identifies enabled user accounts with no recorded last logon date.
These
accounts can indicate incomplete provisioning, abandoned onboarding, or dormant
accounts that were never validated or used.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtAdUserNeverLoggedInCount
```
