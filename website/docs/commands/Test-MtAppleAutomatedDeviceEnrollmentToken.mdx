## SYNOPSIS

Check the validity of the Apple Automated Device Enrollment (ADE) token for Intune.

## SYNTAX

```powershell
Test-MtAppleAutomatedDeviceEnrollmentToken [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

The Apple Automated Device Enrollment (ADE) token is required to synchronize Apple devices with Microsoft Intune.
This command checks if the ADE token is valid and not expired.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtAppleAutomatedDeviceEnrollmentToken
```

Returns true if the ADE token is valid for more than 30 days, false if it is expired or expiring soon.
