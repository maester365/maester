## SYNOPSIS

Checks state of DMARC records for all exo second level domains

## SYNTAX

```powershell
Test-MtCisaDmarcRecordExist [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

A DMARC policy SHALL be published for every second-level domain.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtCisaDmarcRecordExist
```

Returns true if DMARC record exists for all 2LD
