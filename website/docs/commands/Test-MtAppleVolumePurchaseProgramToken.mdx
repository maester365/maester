## SYNOPSIS

Check the validity of the Apple Volume Purchase Program (VPP) token for Intune.

## SYNTAX

```powershell
Test-MtAppleVolumePurchaseProgramToken [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

The Apple Volume Purchase Program (VPP) token is required to synchronize Apple store apps with Microsoft Intune.
This command checks if the VPP token is valid and not expired.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtAppleVolumePurchaseProgramToken
```

Returns true if the VPP token is valid for more than 30 days, false if it is expired or expiring soon.
