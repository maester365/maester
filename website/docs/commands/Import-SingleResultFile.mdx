## SYNOPSIS

Helper: loads a single JSON file and adds valid results to the list.
Handles both single-tenant and multi-tenant (merged) formats.

## SYNTAX

```powershell
Import-SingleResultFile [[-FilePath] <String>] [[-RequiredProperties] <String[]>]
 [[-ResultList] <System.Collections.Generic.List`1[System.Management.Automation.PSObject]>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```
