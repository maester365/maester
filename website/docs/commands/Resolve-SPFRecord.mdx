## SYNOPSIS

Returns a list of all IP addresses from an SPF record

## SYNTAX

```powershell
Resolve-SPFRecord [-Name] <String> [[-Server] <String>] [-Referrer <String>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```
