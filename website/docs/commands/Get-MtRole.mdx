## SYNOPSIS

Returns all the role definitions in the tenant.

## SYNTAX

```powershell
Get-MtRole [-CisaHighlyPrivilegedRoles] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Returns all the role definitions in the tenant.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-MtRole
```
