Checks state of mailbox auditing

## SYNTAX

```powershell
Test-MtCisaMailboxAuditing [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Mailbox auditing SHALL be enabled.

## EXAMPLES

### EXAMPLE 1

Returns true if mailbox auditing is enabled.
