## SYNOPSIS

This test checks if you have any third party service principals that are open to all users.
It is recommended to set 'Assignment required?' to Yes for all Third Party apps.

## SYNTAX

```powershell
Test-MtServicePrincipalsForAllUsers [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Open all app service principals below and set 'Assignment required?' to Yes.
Assign users under 'Users and groups' to provide them with explicit access.
If desired, use the audit logs per SPN to determine who was using the application before locking them down.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtServicePrincipalsForAllUsers
```

Returns true if no third party service principals are assigned to All Users, false if any are found.
