## SYNOPSIS

Checks if all Recovery Services Vaults have Soft Delete enabled

## SYNTAX

```powershell
Test-MtVaultSoftDelete [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

This test ensures that Soft Delete is enabled, Soft Delete ensures that backup items and recovery points are retained for a period after deletion. 
This is important to protect against accidental or malicious deletion of backups.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtVaultSoftDelete
```

Returns true if all Recovery Services Vaults across all subscriptions have soft delete enabled.
