# Apply Streamlined Connectivity to Intune/DFC should be enabled

## Description

Manual review required. Verifies that Streamlined Connectivity is applied to Intune and Defender for Cloud (DFC) for synchronized connectivity. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Apply Streamlined Connectivity to Intune/DFC.
