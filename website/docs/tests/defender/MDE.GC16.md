# Authenticated Telemetry should be reviewed for privacy compliance

## Description

Manual review required. Verifies that Authenticated Telemetry settings comply with organizational privacy requirements. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Review Authenticated Telemetry settings for privacy compliance.
