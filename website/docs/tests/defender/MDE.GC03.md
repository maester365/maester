# EDR in Block Mode should be enabled for Defender AV devices

## Description

Manual review required. Verifies that EDR in Block Mode is enabled so Defender for Endpoint can block malicious artifacts even when a third-party AV is running. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable EDR in Block Mode.
