# Isolation Exclusion Rules should be disabled unless required

## Description

Manual review required. Verifies that Isolation Exclusion Rules are disabled unless specifically required by business justification. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Disable Isolation Exclusion Rules unless business justification exists.
