# Download Quarantined Files should be enabled for forensics

## Description

Manual review required. Verifies that the ability to download quarantined files is enabled for forensic analysis. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Download Quarantined Files.
