# Web Content Filtering should be enabled

## Description

Manual review required. Verifies that Web Content Filtering is enabled. Requires Defender for Endpoint P2 license. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Web Content Filtering (requires P2 license).
