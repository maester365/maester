# Cloud Protection should be enabled

## Description

Verifies that cloud-delivered protection is enabled for real-time threat intelligence from Microsoft cloud.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus. Ensure 'Allow Cloud Protection' is set to 'Allowed'.
