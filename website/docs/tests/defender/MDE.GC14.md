# Deception capabilities should be evaluated

## Description

Manual review required. Verifies that deception capabilities (honeypots) have been evaluated as part of the security strategy. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Evaluate and optionally enable Deception capabilities based on your security strategy.
