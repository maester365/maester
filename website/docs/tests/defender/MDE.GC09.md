# Device Discovery should be enabled for Shadow IT visibility

## Description

Manual review required. Verifies that Device Discovery is enabled to provide visibility into unmanaged devices. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Device Discovery.
