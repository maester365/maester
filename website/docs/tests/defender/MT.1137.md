# Cloud Extended Timeout should be 30-50 seconds

## Description

Verifies that cloud extended timeout is set between 30-50 seconds for optimal suspicious file detection.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus. Set 'Cloud Extended Timeout' to a value between 30 and 50.
