# Custom Network Indicators should be enabled

## Description

Manual review required. Verifies that Custom Network Indicators are enabled for IOC-based network blocking. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Custom Network Indicators.
