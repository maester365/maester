# Microsoft Intune Connection should be enabled as MDM prerequisite

## Description

Manual review required. Verifies that the Microsoft Intune Connection is enabled, which is a prerequisite for MDM integration. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Microsoft Intune Connection.
