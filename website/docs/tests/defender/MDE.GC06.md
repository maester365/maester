# Hide Duplicate Device Records should be enabled

## Description

Manual review required. Verifies that duplicate device records are hidden to reduce clutter in the device inventory. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Hide Duplicate Device Records.
