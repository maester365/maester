# Streamlined Connectivity should be enabled

## Description

Manual review required. Verifies that Streamlined Connectivity is enabled as the default connectivity configuration. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Ensure Streamlined Connectivity is enabled.
