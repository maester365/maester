# Sample Submission should send safe samples automatically

## Description

Verifies that sample submission consent is configured to send safe samples automatically to Microsoft for analysis.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus. Set 'Submit Samples Consent' to 'Send safe samples automatically'.
