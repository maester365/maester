# Email Scanning should be enabled

## Description

Verifies that email scanning is enabled to protect Exchange mail queues from malware.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus. Ensure 'Allow Email Scanning' is set to 'Allowed'.
