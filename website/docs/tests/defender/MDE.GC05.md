# Allow or Block File capability should be enabled

## Description

Manual review required. Verifies that the Allow or Block File capability is enabled for indicator of compromise (IOC) handling. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Enable Allow or Block File.
