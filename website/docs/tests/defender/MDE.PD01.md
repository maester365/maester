# Policy naming should follow consistent convention (ROLE-v#)

## Description

Manual review required. Verifies that MDE policies follow a consistent naming convention such as AV-PL-Client-Gen-v1. This is a manual review test.

## How to fix

1. Navigate to Microsoft Intune admin center (intune.microsoft.com).
2. Go to Devices > Configuration.
3. Review all Microsoft Defender Antivirus policies.
4. Ensure naming follows a consistent convention (e.g., AV-PL-Client-Gen-v1, AV-EX-Server-DB-v2).
