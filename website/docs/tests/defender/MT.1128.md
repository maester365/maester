# Real-time Monitoring should be enabled

## Description

Verifies that real-time protection monitoring is enabled for live threat detection.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus > Real-time Protection. Ensure 'Allow Realtime Monitoring' is set to 'Allowed'.
