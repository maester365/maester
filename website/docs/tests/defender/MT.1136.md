# Cloud Block Level should be High or higher

## Description

Verifies that the cloud block level is set to 'High' or higher for maximum cloud-based protection.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus. Set 'Cloud Block Level' to 'High' or 'High Plus' or 'Zero Tolerance'.
