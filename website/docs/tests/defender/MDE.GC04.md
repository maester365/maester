# Automatically Resolve Alerts should be configured

## Description

Manual review required. Verifies that automatic alert resolution is properly configured. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Endpoints > Advanced Features.
3. Configure Automatically Resolve Alerts as appropriate.
