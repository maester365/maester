# Scan should be scheduled

## Description

Verifies that a scheduled scan day is configured.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus > Scan. Configure 'Schedule Scan Day' to a specific day of the week.
