# Preview Features should be enabled organization-wide

## Description

Manual review required. Verifies that Preview Features are enabled organization-wide in the Microsoft Defender XDR portal. This is a manual review test.

## How to fix

1. Navigate to Microsoft Defender XDR portal (security.microsoft.com).
2. Go to Settings > Microsoft Defender XDR > Preview Features.
3. Enable Preview Features for the organization.
