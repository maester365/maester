# PUA Protection should be enabled

## Description

Verifies that Potentially Unwanted Application (PUA) protection is enabled.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus. Ensure 'PUA Protection' is set to 'PUA Protection on'.
