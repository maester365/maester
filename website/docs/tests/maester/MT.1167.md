# Real-Time Scan Direction should cover both directions

## Description

Verifies that real-time scan direction is set to monitor both incoming and outgoing files.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus > Real-time Protection. Ensure 'Real Time Scan Direction' is set to 'Monitor all files (bi-directional)'.
