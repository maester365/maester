# Signature Update Interval should be 1-4 hours

## Description

Verifies that the signature update interval is set between 1-4 hours for timely definition updates.

## How to fix

Navigate to Intune > Devices > Configuration > Settings Catalog > Microsoft Defender Antivirus > Signature Updates. Set 'Signature Update Interval' to a value between 1 and 4.
